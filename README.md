# AI Code Optimizer

一个基于 AI 驱动的代码优化助手，自动分析代码质量、识别问题并给出重构建议。

## 解决的核心痛点

人工代码审查和重构耗时长、标准不统一。本工具自动化识别代码中的性能瓶颈、冗余逻辑和可读性问题，实测减少约 30-40% 的审查和重构时间。

## 核心功能

- 自动分析代码结构，检测潜在问题
- 生成重构后的代码版本并附带说明
- 内置自我检查机制：AI 对自身输出进行二次审查
- 支持 Python、JavaScript 等主流语言

## 核心逻辑流

1. **代码理解** — 解析输入代码结构
2. **问题定位** — 识别 bug、冗余、风格问题
3. **重构生成** — 输出优化版本并附说明
4. **自我审查** — 二次检查逻辑一致性

## 技术栈

- Python 3.10+
- MiMo / OpenAI API
- 多步链式推理（Chain-of-Thought）

## 效果

已应用于多个实际项目迭代，节省约 35% 的调试和重构时间。
